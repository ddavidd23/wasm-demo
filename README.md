# wasm-demo
WaSM Demo for CPSC 4210

## VS-Code instructions
1. Update `c_cpp_properties.json` with `"compilerPath": {which em++}` and appending full path of `wasm/emsdk/upstream/emscripten/system/include` and `wasm/emsdk/upstream/emscripten/cache/sysroot/include` to `"includePath"`
2. Reset IntelliSense Database and reload window
